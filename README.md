# Prediction-of-Life-Expectancy-Using-Socio-Economic-Indicators-A-Statistical-and-ML-Approach
This project analyzes and predicts life expectancy using secondary data from the World Health Organization. Statistical and machine learning models such as linear regression and random forest are applied in Python to identify key socio-economic and health factors influencing life expectancy.
